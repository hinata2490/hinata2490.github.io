自己紹介

はじめまして、棚橋広翔です。動画編集・デザインを中心に、 AIを活用した制作フローで、品質とスピードの両立を目指しています。 丁寧なヒアリングから構成・編集まで一貫して対応し、 ブランドやサービスの価値が伝わるアウトプットを大切にしています。

できること

  • 動画編集（企画整理 / カット / テロップ / BGM・効果音 / 演出調整）
  • デザイン制作（サムネイル / バナー / SNS投稿画像 / KV）
  • AI活用（リサーチ補助 / 構成案作成 / 制作効率化）
