# TikTokコスメ擬人化チャンネル「コスメの事情」 - プロジェクト概要

## このプロジェクトは何か

化粧品（ボトル）が顔と手足を持つPixar風3DCGキャラとして登場し、感情むき出しで使い方や勘違いをキレながら解説するTikTokエンタメチャンネル。

YouTube側（睡眠解説「レムの睡眠ラボ」）とは別プロジェクト。素材・キャラ・パイプラインは共有しない。

## オーナーについて

- プログラミング初心者（API・Git等は Claude Code 任せ）
- 動画編集はプロ。Premiere Pro / After Effects / 編集案件 600件以上の納品実績
- **最大の参入障壁は編集力** という認識。台本・カット・テンポ・テロップ感覚で勝負する
- 操作手順はステップバイステップ・コピペ実行できる形で出すこと

## 戦略の核心（一言で）

> 実写でバズってる擬人化美容アカウントが既にある。AI生成で制作コストをゼロにし、編集力で品質差をつける。

詳細は [docs/strategy.md](docs/strategy.md)。

## チャンネル基本仕様

- **アカウント**: コスメの事情（@cosme_no_jijou）
- **ターゲット**: 18〜28歳女性、美容初心者層
- **トーン**: 怒り→落ち着いて解説→また怒り、の緩急
- **動画尺**: 30〜45秒（フォロワー1万超で1分超に切替し広告収益も取る）
- **投稿頻度**: 週5本
- **編集**: フルテロップ、ナレーションなし、全部キャラのセリフ
- **冒頭0.5秒で掴む**

## キャラデザインルール

- 商品ボトルに顔と手足がついた Pixar風 3DCGキャラ
- **ロゴは再現しない**（商標権回避）
- ボトル形状・色味・サイズ感で「あの商品だな」とわかる程度に特徴を残す
- 商品名のセリフ・テロップ言及はレビュー範疇でOK

## 制作ワークフロー（順番厳守）

1. **Claude** で台本作成（このリポジトリ内 `episodes/` に保存）
2. **Gemini** でキャラ画像生成（参照画像渡す、ロゴなし）
3. **CapCut** で感情付き音声生成（1.2倍速）
4. 各セリフの秒数を計測
5. **Kling** で各カット動画生成（720p、1カット10秒以内、余白なし）
6. **Kling** でリップシンク合成
7. **Premiere Pro** で仕上げ（クロマキー / テロップ / カット / テンポ / BGM・SE）

詳細は [docs/production.md](docs/production.md)。

## Kling運用ルール（コスト最適化）

- 720pで生成（1080pは1本あたり約240クレジット余分）
- 1カット10秒以内（リップシンク上限）
- 1カットにつきリテイク2〜3回想定
- 背景はKlingで指定せず、Premiereでクロマキー合成

## 月額コスト合計

## 法務・コンプライアンス

- ロゴ・ブランド名の画像再現は **しない**
- AI生成ラベル表示は **必須**
- 美容品擬人化はフィクションなのでTikTokのディープフェイク制限には該当しない
- 広告収益（Creator Rewards Program）はAI動画だと収益レートが下がる可能性あり → アフィリエイト主軸で設計
