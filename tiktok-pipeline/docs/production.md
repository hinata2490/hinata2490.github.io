# 制作

## 動画仕様

| 項目 | 値 |
|---|---|
| 尺 | 30〜45秒（フォロワー1万超で1分超に切替） |
| 解像度 | 720p（Kling生成時） |
| 編集 | フルテロップ、ナレーションなし、全部キャラのセリフ |
| 投稿頻度 | 週5本 |
| 冒頭 | 0.5秒で掴む |

## 制作ワークフロー（5ステップ）

```
① Claude（台本）
  ↓
② Gemini（キャラ画像生成、参照画像渡す、ロゴなし、グリーンバック、全身）
  ↓
③ CapCut（感情付き音声生成、1.2倍速）
  ↓
④ CapCut（リップシンク：キャラ画像1枚 + 音声 → 動画化＆全フレーム口パク同時生成）
  ↓
⑤ Premiere Pro（クロマキー合成、テロップ、カット、テンポ調整、BGM・SE）
```

## パイプライン設計の経緯（2026-04-18）

当初は Gemini → Kling動画生成 → Klingリップシンク → Premiere の構成だった。検証中に判明：

- **Kling のリップシンク**は「最適吹き替え区間」（口元が安定して認識できるフレーム）のみに適用され、動画全体に効かない
- **CapCut のリップシンク**は画像1枚 + 音声を渡すと、**動画自体を生成 + 全フレーム口パク**を一発でやってくれる
- CapCutの動画品質はKlingと大差なし（短尺TikTok用途の場合）
- Klingコスト $65/月 が浮く

→ Kling を外し CapCut 中心に変更。

## CapCut 運用ルール

- **入力**: Geminiで作ったキャラ画像（グリーンバック・全身）+ CapCutで作った音声mp3
- **出力**: 動画（口パク済み）
- **モーション制御**: プロンプト指定不可（自動生成のおまかせ）
- **動きの品質**: 結構動く。トーキングヘッド系の用途なら十分
- **コスト**: AI機能（リップシンク等）はCapCut Pro契約とは別にクレジット課金。Klingより安い

## ⚠️ CapCut の表情制約と対処（A案：Gemini表情画像方式）

CapCutリップシンクは **キャラの怒り表情を自然な喋り顔に均してしまう傾向** があり、入力画像が怒り顔でも結果は中立的になりやすい。

「コスメの事情」は**怒り表現が看板**なので、ep02以降は以下の方式を採用：

1. Geminiで**表情バリエーション画像**を事前生成
   - 怒り顔（眉吊り上げ・口開き）
   - 呆れ顔（半目・下がり眉）
   - ドヤ顔（自信満々・笑み）
2. カットごとに**最も合う表情の画像をCapCutに入力**
3. 入力時の表情が結果動画に**多少残る**ことを期待

完全な表情制御はできないが、現状のベストプラクティス。

### 検証済み（ep01・2026-04-18）
ep01は**怒り表情なし版**で投稿。表情問題の解決はep02からの課題。

## Kling は将来オプション

派手なモーション・カメラワーク・複雑なアクション動画が必要になったタイミングで再契約検討。現状の「キャラがその場で喋ってジェスチャーする」用途には不要。

## キャラデザイン制約

- 商品ボトルに顔と手足がついた **Pixar風3DCGキャラ**
- **ロゴは再現しない**（商標権回避）
- ボトル形状・色味・サイズ感で「あの商品だな」とわかる程度に特徴を残す

## 著作権・商標権

| 項目 | 方針 |
|---|---|
| 商品パッケージのロゴ・ブランド名の画像再現 | **しない** |
| ボトルの形状・色味・サイズ感 | 「あの商品だな」とわかる程度に留める |
| セリフ・テロップでの商品名言及 | レビュー・コメント範囲でOK（TikTok公式ポリシーでも許容） |
| AI生成ラベル表示 | **必須** |

## TikTokのAI動画制限

- AI動画の投稿自体は問題なし（ラベルつければOK）
- 禁止されてるのは**ディープフェイクや誤解を招くリアルなAI動画**
- 美容品擬人化はフィクションなので該当しない
- 広告収益（Creator Rewards Program）はAI動画だと**収益レートが下がる可能性あり**
- TikTok Shopアフィリエイトにはこの制限は適用されない

→ **アフィリエイト主軸で設計する**理由がここにある

## 編集方針（2026-04-18 方針転換）

凝った演出は **やらない**。AI動画系のバズ動画はほぼテロップだけで成立しており、編集の費用対効果は急速に逓減する。

### やる
- クロマキー合成
- テロップ（パッと切替・フェード無し）
- BGM音量バランス
- 要所のSE（1本あたり3〜4個）
- 基本カット割り（テンポ）

### やらない
- テロップアニメーション
- ズーム演出
- トランジションエフェクト
- After Effects 作業

### 編集スキルを投下する4つの判断ポイント
編集力は「演出を盛る方向」じゃなくて「判断の精度」で発揮する。

1. **どこでセリフをぶった切るか**（テンポ）
2. **どこで一瞬無音にするか**（間で笑わせる）
3. **テロップの「乗せる位置」と「消すタイミング」**
4. **冒頭フックの構図設計**（0:00の絵を何秒キープするか）

これは編集経験ゼロの人にはできない。一般人/AI素人との差は十分出る。

### 1本あたりの編集時間目標
**60分以内**。これを超えたら週5本量産に間に合わない。
