## 収益導線

### ① TikTok Shop アフィリエイト

- **条件**: フォロワー1,000人以上
- アプリ内マーケットプレイスから商品選ぶだけ。在庫・発送不要
- **全動画にカートをつけない**（7割エンタメ、3割購買導線）
- バズった動画に30日以内なら後付けでカート追加可能
- 報酬率: 商品により5〜20%、高いものは80%超

### ② PR案件

- **条件**: フォロワー5万〜
- 相場: フォロワー数×2〜3円（企業側）、手取りは×0.5〜2円
- 美容系は案件数が多いジャンル

### ③ 広告収益（Creator Rewards Program）

- **条件**: フォロワー1万以上 + 1分以上の動画
- 1再生あたり約 0.02〜0.07円
- フォロワー1万超えてから動画を1分超に切り替えて取りにいく

## 収益シミュレーション

- 全動画平均CVR: 0.1〜0.15%が現実的
- TikTok Shop単体CVR（カート付き動画クリック後）: 5〜8%が良い水準
- 再生→クリック率: 0.5%以上が目安

## 編集方針の方針転換（2026-04-18）

当初は「編集力が真の参入障壁」と位置づけていたが、検証の結果これは過大評価と判断。理由：

- AI動画系の参考バズ動画は**ほぼテロップだけ**で成立している
- 編集の費用対効果は急速に逓減：「編集ゼロ→テロップ追加」で視聴維持率の80%が取れる
- 凝った演出（テロップアニメ・ズーム・AE作業）の追加効果は微々たるもの
- 週5本の量産には**1本60分以内の編集スピード**が必要

### 新しい編集方針

| やる | やらない |
|---|---|
| クロマキー合成 | テロップアニメーション |
| テロップ（パッと切替・フェード無し） | ズーム演出 |
| BGMの音量バランス | トランジションエフェクト |
| 要所のSE（3〜4個） | After Effects作業 |
| 基本カット割り（テンポ） | 凝った構図演出 |

### 編集力を投下するべき4つの判断
1. **どこでセリフをぶった切るか**（テンポ）
2. **どこで一瞬無音にするか**（間で笑わせる）
3. **テロップの「乗せる位置」と「消すタイミング」**
4. **冒頭フックの構図設計**（0:00の絵を何秒キープするか）

→ 編集スキルゼロの人とはここで差がつく。「アニメーション盛る」じゃなくて「**判断の精度**」で差別化。

### 浮いた時間の使い道
- 1本あたりの制作時間が大幅短縮 → 量産可能
- 余った時間は **次の台本3本作る** に投下する方がROI高い

## なぜこの戦略が成立するか（面接用ロジック）

1. 市場の需要は実写で証明済み。実写でバズってる台本構成をAI擬人化に変換するだけ
2. AIが2つの変化を起こした: **制作コストの消滅** + **実写では不可能な表現の実現**
3. 競合がほぼゼロ。先行者が2ヶ月で4.2万フォロワーという成長速度が「需要」と「競合不在」を同時に証明
4. 美容系はTikTok Shop売上の約 **22.4%** を占めるトップカテゴリ
5. **編集スキルは「演出を盛る方向」ではなく「判断の精度」に効く**。テンポ・間・フック構図の判断は経験値でしか出せない、でも凝った演出は不要。これにより**量と質を両立**できる
6. 話の主語は「AI」ではなく **「市場機会の発見と参入判断」**
