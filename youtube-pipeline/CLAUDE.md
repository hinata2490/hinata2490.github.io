## 動画品質の基準（編集マニュアルより抜粋・重要部分）
オーナーの案件用編集マニュアル（39ステップ）から、自動パイプラインに適用する部分：

### 優先する品質基準
- **カット**: 無音部分・言い直し・重複箇所をしっかり除去
- **音量**: 声の音量を統一。BGM・SEはリミッター -20dB
- **テロップ**: 表示されない瞬間をゼロにする。誤字脱字・表記揺れゼロ
- **テロップ位置**: 底辺を揃える。必ず中央配置
- **表示時間**: 短すぎるものは残さない

### 一旦やらないこと（コストが高すぎるため）
- テキストアニメーション（Remotionで実装すると複雑すぎる）
- SE・エフェクト演出
- 6秒に1回の演出ルール
- 画像挿入アニメーション

## 動画構成フォーマット
1. **冒頭**: よくある悩み提示（「こんなことで困っていませんか？」）または睡眠の時事ネタ
2. **解決策**: 一般的な解決策を先に提示
3. **具体例**: 詳しく解説
4. **まとめ**: 要点を再提示
5. **エンディング**: チャンネル登録呼びかけ

※煽り・強い言葉はチャンネルコンセプトに合わないためNG。

## パイプライン構成（6ステップ全自動）
1. **collect** - ネタ収集（RSS/YouTubeトレンド → Claudeで優先順位付け）
2. **script** - 台本生成（Claude APIで10分台本を自動生成）
3. **audio** - 音声合成（VOICEVOX → OpenAI TTS → gTTS のフォールバック）
4. **video** - 動画合成（Remotionベース）
5. **thumbnail** - サムネイル作成（Pillowで自動生成）
6. **upload** - YouTube投稿（OAuth認証で自動アップロード）

## 実行方法
```bash
# 全ステップ実行
python pipeline.py

# 特定トピックで実行
python pipeline.py --topic "睡眠の質を上げる方法"

# 特定ステップから再開
python pipeline.py --from-step audio

# テスト実行（API呼ばない）
python pipeline.py --dry-run
```
